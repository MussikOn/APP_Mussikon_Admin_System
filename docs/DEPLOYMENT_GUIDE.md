### **1. Script de Rollback**

```bash

# rollback.sh

set -e

echo "üîÑ Iniciando rollback..."

# Variables

BACKUP_DIR="/var/backups/mussikon-admin"

    echo "‚ùå Backup no encontrado"
    exit 1

# Crear backup del estado actual
echo "üì¶ Creando backup del estado actual..."

# Restaurar backup
echo "üîÑ Restaurando backup..."
sudo rm -rf $APP_DIR/dist
sudo tar -xzf $BACKUP_DIR/backup_$BACKUP_TIMESTAMP.tar.gz -C $APP_DIR

# Reiniciar aplicaci√≥n
echo "üîÑ Reiniciando aplicaci√≥n..."
pm2 restart ecosystem.config.js
```
